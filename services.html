METALLURGICAL CORED MACHINE

Metallurgical cored machine installation requires 20 meter (width) x 40 meter (length) x10meter (height) space for one line. Raw material for producing cored wire is depends up on product.

  • Generally 0.4mm thickness and 58mm width CRC EDD grade strip as per IS 513 is require for 13mm diameter.
  • 0.7mm thickness and 36mm width strip for solid calcium.
  • 0.4mm thickness and 41mm width for 9mm dia powder wire.
  • 0.4mm thickness and 70mm width for 16mm wire.

    • Machine is suitable to produce form 9mm diameter to 20mm diameter cored wire.

    As per metallurgical requirement different cored wire can be manufactured through different powder composition.
